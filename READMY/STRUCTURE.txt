ДЕТАЛЬНОЕ ОПИСАНИЕ СТРУКТУРЫ:

READMY/ - содержит всю документацию проекта

READMY_MAIN.txt: полное описание функциональности, архитектуры и использования

STRUCTURE.txt: иерархия файлов и папок проекта

UI/ - исходные файлы интерфейса Qt Designer

GUI_FREL.ui: XML-описание главного окна с разметкой всех элементов

CreateNPY_FREL.ui: XML-описание окна создания базы данных

Основные модули Python:

GUI_FREL.py: основной класс приложения, управление интерфейсом и потоками

AI_YOLO_FREL.py: ядро обработки - детекция, трекинг и распознавание

TheFace_FREL.py: логика работы с базой данных лиц и сравнения сигнатур

CreateNPY_FREL.py: функционал создания и объединения баз данных

LinerDasher_FREL.py: вспомогательные функции графики

Сгенерированные UI модули (не редактировать вручную):

UI_GUI_FREL.py: автоматически сгенерированный из GUI_FREL.ui

UI_CreateNPY_FREL.py: автоматически сгенерированный из CreateNPY_FREL.ui

Ресурсы:

yolov8n-face.pt: предобученная модель YOLO для обнаружения лиц

face_database.npy: бинарный файл с базой данных лиц (создается при первом запуске)

data/: папка с каскадами Haar для OpenCV

Вспомогательные файлы:

config.py: настройка переменных окружения и путей

hook-tools_all.py: конфигурация хуков для корректной сборки PyInstaller

build.txt: параметры и скрытые импорты для сборки

ЗАВИСИМОСТИ МЕЖДУ МОДУЛЯМИ:

GUI_FREL.py → UI_GUI_FREL.py (интерфейс)
GUI_FREL.py → AI_YOLO_FREL.py (обработка видео)
GUI_FREL.py → CreateNPY_FREL.py (управление БД)
GUI_FREL.py → TheFace_FREL.py (распознавание)

AI_YOLO_FREL.py → TheFace_FREL.py (сравнение лиц)
AI_YOLO_FREL.py → LinerDasher_FREL.py (визуализация)

CreateNPY_FREL.py → UI_CreateNPY_FREL.py (интерфейс БД)

ВСЕ МОДУЛИ → config.py (настройки окружения)

СБОРКА ПРОЕКТА:
Проект использует PyInstaller для создания исполняемого файла.
Конфигурация сборки описана в build.txt и hook-tools_all.py

ЗАПУСК:
Основная точка входа - GUI_FREL.py
Для разработки: python GUI_FREL.py
Для production: использовать собранный исполняемый файл
