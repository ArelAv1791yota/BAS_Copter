BAS_ARELERIO - Система детекции и распознавания лиц
ОБЩЕЕ ОПИСАНИЕ
BAS_ARELERIO - это комплексная система компьютерного зрения для обнаружения, отслеживания и распознавания лиц в реальном времени. Проект использует современные технологии машинного обучения и компьютерного зрения для обработки видеопотоков.

ОСНОВНЫЕ ВОЗМОЖНОСТИ
Детекция лиц в реальном времени с помощью YOLOv8-face

Распознавание лиц на основе гистограмм HSV

Отслеживание траекторий движения лиц

Обработка видеофайлов в фоновом режиме

Настройка фильтров изображения (яркость, контраст, резкость)

Создание и объединение баз данных лиц

АРХИТЕКТУРА СИСТЕМЫ
1. ГЛАВНОЕ ОКНО (GUI_FREL.py)
Основной модуль приложения, содержащий графический интерфейс и управляющую логику.

Основные компоненты:

MainWindow - главное окно приложения

TargetWidget - виджет для отображения информации об обнаруженных лицах

Обработчики видеопотоков (камера, видеофайлы)

Управление фильтрами изображения

Режимы работы:

Режим камеры - обработка видеопотока с веб-камеры

Режим воспроизведения - простой просмотр видеофайлов

Режим обработки - детекция и распознавание лиц в видеофайлах

Фоновая обработка - пакетная обработка видео с сохранением результатов

2. ДЕТЕКЦИЯ И РАСПОЗНАВАНИЕ (AI_YOLO_FREL.py)
Ядро системы обработки изображений.

Классы:

FaceDetector - детекция лиц с использованием YOLOv8 и управление траекториями

VideoProcessor - обработка видеофайлов в фоновом режиме

Особенности:

Использование GPU через PyTorch при наличии

Адаптивное управление FPS для оптимальной производительности

Система трекинга лиц с уникальными ID

История распознаваний для улучшения стабильности

3. РАСПОЗНАВАНИЕ ЛИЦ (TheFace_FREL.py)
Модуль для создания и сравнения сигнатур лиц.

Класс FaceRecognitionIntegrator:

Загрузка и управление базой данных лиц

Создание сигнатур на основе гистограмм HSV

Сравнение сигнатур с использованием корреляции

Система majority voting для стабильного распознавания

4. СОЗДАНИЕ БАЗЫ ДАННЫХ (CreateNPY_FREL.py)
Инструмент для создания и управления базой данных лиц.

Классы:

DatabaseCreationThread - поток для создания БД

CreateNPYWindow - интерфейс для управления БД

Режимы создания:

Из папки с изображениями

С использованием веб-камеры

Комбинированный режим

5. УТИЛИТЫ (LinerDasher_FREL.py)
Вспомогательные функции для визуализации.

Класс DrawingUtils:

Рисование пунктирных линий и рамок

Визуальное выделение распознанных лиц

ФАЙЛЫ КОНФИГУРАЦИИ И РЕСУРСЫ
Модели:
yolov8n-face.pt - предобученная модель YOLOv8 для детекции лиц

Базы данных:
face_database.npy - основная база данных лиц (создается автоматически)

Каскады Haar:
haarcascade_frontalface_default.xml - для обнаружения лиц при создании БД

ИНТЕРФЕЙС ПОЛЬЗОВАТЕЛЯ
Главное окно разделено на две части:
Левая панель (300px):

Настройки фильтров (яркость, контраст, резкость)

Информация о видеопотоке (разрешение, FPS)

Список обнаруженных целей с координатами

Управляющие кнопки

Правая панель (1280px):

Отображение видеопотока

Визуализация обнаруженных лиц и траекторий

Панель управления видео (слайдер, кнопки управления)

Элементы управления:
DB - открытие окна управления базой данных

Сброс - сброс фильтров к значениям по умолчанию

Кнопки управления видеопотоками

Слайдеры настройки фильтров

РАБОТА С БАЗОЙ ДАННЫХ ЛИЦ
Создание базы:
Откройте окно "Создание базы данных лиц"

Выберите режим создания (папка, камера или комбинированный)

Укажите имя для базы данных

Запустите процесс создания

Формат базы данных:
База сохраняется в формате NPY и содержит словарь, где:

Ключи - имена людей

Значения - списки гистограмм HSV для каждого образца

ТЕХНОЛОГИИ И БИБЛИОТЕКИ
Основные:
PySide6 - графический интерфейс

OpenCV - обработка изображений и видео

PyTorch - машинное обучение

Ultralytics YOLOv8 - детекция объектов

NumPy - численные вычисления

Дополнительные:
SciPy - научные вычисления

scikit-learn - алгоритмы машинного обучения

Matplotlib - визуализация (для отладки)

Требования:
Python 3.8+

PyTorch с поддержкой CUDA (рекомендуется)

Современная веб-камера (для режима реального времени)

Запуск:
bash
python GUI_FREL.py
Сборка в исполняемый файл:
